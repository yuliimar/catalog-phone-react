# Catalog Phone React

A responsive e-commerce product catalog built with React and TypeScript, featuring shopping cart functionality, favorites management, and theme switching capabilities.

## 🚀 Demo

🔗 [Live Preview](https://yuliimar.github.io/catalog-phone-react/)

## 🎨 Design Reference

🖼️ [Figma Mockup - Original Dark Theme](https://www.figma.com/design/BUusqCIMAWALqfBahnyIiH/Phone-catalog--V2--Original-Dark?node-id=0-1&p=f&t=Rq9jWsDp7mebYDEi-0)

## 🛠️ Technologies Used

- **Frontend Framework**: React 18, TypeScript
- **Styling**: SCSS Modules, CSS Variables
- **Routing**: React Router DOM
- **State Management**: React Context API
- **Build Tool**: Vite
- **Deployment**: GitHub Pages
- **Code Quality**: ESLint, Prettier, Husky

## 📦 Getting Started

Follow these steps to set up the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/<your_account>/<repo_name>.git
cd <repo_name>
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the project locally

```bash
npm start
# or
yarn start
```

## ✨ Features

### 🛍️ Core Functionality

Product Catalog - Browse phones, tablets, and accessories

Shopping Cart - Add/remove items with persistent storage

Favorites System - Save products with local storage persistence

Product Details - Comprehensive product information with image gallery

### 🎯 User Experience

Responsive Design - Optimized for all device sizes

Theme Switching - Multiple color themes (Original, Dark, Rounded variants)

Smooth Animations - CSS transitions and hover effects

Accessibility - Semantic HTML and keyboard navigation

### 🔧 Advanced Features

Search & Filter - Find products with debounced search

Sorting & Pagination - Organize products with URL persistence

Product Recommendations - "You may also like" suggestions

Breadcrumb Navigation - Easy navigation hierarchy

### 📱 Pages & Routing

Home - Featured products, categories, and promotional sliders

Product Categories - Dedicated pages for phones, tablets, accessories

Product Details - Comprehensive product information

Shopping Cart - Cart management and checkout simulation

Favorites - Saved products collection

404 Page - Custom not found page

### 🎨 Theme System

Original Dark - Dark mode variant
